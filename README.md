#### Get country name and weather details 

```http
  https://api.openweathermap.org/data/2.5/weather?q=cityName&appid=apiKey
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `q` | `string` | **Required**. Your cityName |
| `apiKey` | `string` | **Required**. Your API key |
